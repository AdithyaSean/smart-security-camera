About
=====

This is documentation of `ESP-WHO <https://github.com/espressif/esp-who>`_, the framework to develop image applications for `ESP32 <https://espressif.com/en/products/hardware/esp32/overview>`_ chip by `Espressif <https://espressif.com>`_.

The **ESP32** is 2.4 GHz Wi-Fi and Bluetooth combo, 32 bit dual core chip running up to 240 MHz, designed for mobile, wearable electronics, and Internet-of-Things (IoT) applications.

With **ESP-WHO**, you can easily build up face detection- and recognition-featured applications. In general, the ESP-WHO features will be supported as shown below:

.. figure:: ../_static/overview.jpg
    :align: center
    :alt: ESP-WHO
    :figclass: align-center

    Espressif Audio Development Framework

The **ESP-WHO** builds on well established, FreeRTOS based, Espressif IOT Development Framework `ESP-IDF <https://github.com/espressif/esp-idf>`_.
